# BI Bourse V5 — Explorateur Financier

## Modules
- Trading avec Lightweight Charts, bougies et ligne.
- Explorateur de produits liés.
- Fiches pédagogiques action, ETF et dérivés.
- Comparateur de quatre produits.
- Assistant pédagogique local.
- Portefeuille virtuel.

## Données
Le flux historique et temps réel est opérationnel pour les cryptos Binance. Les actions sont en mode OHLC simulé tant que le connecteur de données actions n’est pas finalisé.

Les relations ETF et dérivés sont une base pédagogique indicative. Avant utilisation réelle, elles devront être alimentées par les fiches officielles des émetteurs et mises à jour régulièrement.
