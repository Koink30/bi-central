# BI Simulateur Boursier

Prototype de BI pour simuler du boursicotage avec portefeuille virtuel, journal d'ordres, scénarios et connexion à des cours via API.

## Fichiers
- `index.html` : starter déployé dans le repo.
- Package V1 complet généré côté ChatGPT : version plus riche avec connecteurs Alpha Vantage, Twelve Data, FMP et proxy Apps Script.

## Déploiement GitHub Pages
1. Repo : `Koink30/bi-central`.
2. Chemin : `bi-simulateur-boursier/index.html`.
3. Pour publication simple : activer GitHub Pages depuis `main` puis ouvrir le chemin `/bi-simulateur-boursier/`.

## API
Ne jamais publier une clé API dans un repo public. Pour une V2 propre : utiliser un proxy Google Apps Script ou un petit backend.

## Roadmap
- V1 : simulation locale + export JSON.
- V2 : proxy Apps Script + sauvegarde Drive/Google Sheets.
- V3 : backtesting, alertes, règles de risque, fiches pédagogiques.

## Avertissement
Outil pédagogique uniquement. Aucune recommandation d'investissement.
