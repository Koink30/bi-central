# Cockpit Projets BI — V6.8A

Nouvelle version expérimentale créée à partir de l'analyse de la V6.7F.

## Objectif

Transformer le cockpit projets BI en générateur d'offres client modulaires :

- création d'un projet client parent ;
- création automatique des sous-modules BI ;
- reprise visuelle du système projet / sous-objet ;
- intégration des 4 modules client de base : Commerce, Clients & Commandes, Planning & Activité, Facturation ;
- packs d'offres Découverte, Standard, Confort ;
- génération d'une roadmap V0, d'un manifest et d'une structure Drive/GitHub théorique.

## Modules intégrés

### BI Commerce
Produits, prix, étiquettes, commandes, stock simple.

### BI Clients & Commandes
Clients, demandes, commandes, relances, suivi.

### BI Planning & Activité
Planning, tâches, équipe, échéances, vue synthèse.

### BI Facturation
Devis, factures, encaissements, livre recettes, exports.

### Modules bonus Confort
Assistant IA et Drive Sync / Livraison.

## Notes

Cette version est additive et ne remplace pas la V6.7F stable. Elle sert de laboratoire pour valider la logique d'offres avant fusion dans le cockpit principal.
